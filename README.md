# Subscription Service

## Описание

**Subsciption Service** - это простой REST сервис для управления онлайн подписками пользователей

## Запуск

0. Перед запуском убедитесь, что у вас установлен **Docker engine**. Если он не установлен, то можете скачать его [тут](https://www.docker.com/products/docker-desktop/)

1. Клонируйте этот репозитории себе на устройство:
  ```
   git clone https://github.com/DimaKropachev/sub-service 
  ```

2. Перейдите в папку проекта:
  ```
  cd sub-service
  ```

3. Откройте **Docker engine**

4. Запустите сервер:
  ```
docker-compose up -d 
  ```

## Документация API (Swagger)

После запуска приложения интерактивная документация будет доступна по адресу: http://localhost:8083/swagger/index.html

## Конфигурация

Приложение настраивается через переменные окружения. Все параметры имеют значения по умолчанию, поэтому для базового запуска достаточно просто запустить проект.

### Доступные переменные

| Переменная | Описание | Значение по умолчанию |
|------------|----------|----------------------|
| `ENVIRONMENT` | Окружение (dev/prod) | `dev` |
| `HTTP_PORT` | Порт HTTP сервера | `8083` |
| `POSTGRES_USER` | Имя пользователя БД | `postgres` |
| `POSTGRES_PASSWORD` | Пароль пользователя БД | `postgres` |
| `POSTGRES_HOST` | Хост БД | `db` |
| `POSTGRES_PORT` | Порт БД | `5432` |
| `POSTGRES_DB` | Название базы данных | `postgres` |
| `POSTGRES_SSLMODE` | Режим SSL | `disable` |
